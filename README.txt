General Info:

Simple Web App created with React JS and Firebase to store information about favourite games. Every Guest can view all the games and read their info but can not edit or delete anything. Registered Users can Add games and Edit and Delete only their added games.



Technologies:

Project is created with:
React JS version: 17.0.1
React DOM version: 17.0.1
React Router DOM version: 5.2.0
React Scripts version: 4.0.3
Firebase version: 8.3.2
React-Firebase/Auth version: 0.2.10



Setup and Launch:

To run this project, install it locally using npm:
cd client
npm install
npm start
